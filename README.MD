# MemoryLog – Early Stage Dementia Detection

A web app that helps detect early signs of dementia through cognitive memory games.

## What it does
Users play memory-based games like Matching Pairs, Word Recall, and Pattern Sequence. Their scores are tracked over time and shown to caregivers and admins through separate dashboards.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- Database: MongoDB

## Features
- Login & Registration
- 3 Memory Game Modules
- Patient, Caregiver & Admin Dashboards
- Progress Tracking
- REST APIs

## How to Run
1. Clone the repo
2. Run `pip install flask pymongo`
3. Run `python app.py`
4. Open `http://localhost:5000`
